# mcp-server-schedule-manage

A simple MCP schedule manage server.

## 安装
```bash
pip install mcp-server-schedule-manage
```

## 用法

### 命令行

安装后可用 `xtsm` 命令（如有配置）：
```bash
xtsm
```

### 作为库导入

```python
from mcp-server-schedule-manage import main
main()
```
