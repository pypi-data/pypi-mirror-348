# A2A SDK (Prototype)

Early version of the A2A Python SDK.

## Install SDK

```bash
pip install .
```

## Verify install

```py
import a2a
```

## Run Helloworld

Run Remote Agent

```bash
cd examples/helloworld
python __main__.py
```

In another terminal, run the client

```bash
python test_client.py
```
